# Todo List Interactive

Ce projet est une Todo List interactive développée en HTML, CSS et JavaScript.

Il a été réalisé dans le cadre de mon apprentissage du développement web, afin de pratiquer la manipulation du DOM, les événements JavaScript, les tableaux d'objets et la sauvegarde de données avec le localStorage.

## Objectif du projet

L'objectif de ce projet est de créer une application simple permettant de gérer une liste de tâches directement depuis le navigateur.

Ce projet m'a permis de consolider les bases de JavaScript tout en travaillant sur une interface utilisateur claire et fonctionnelle.

## Fonctionnalités

- Ajouter une nouvelle tâche
- Refuser l'ajout d'une tâche vide
- Modifier le texte d'une tâche
- Supprimer une tâche
- Changer le statut d'une tâche :
    - Non réalisée
    - En cours
    - Terminée
- Filtrer les tâches selon leur statut
- Afficher des statistiques sur les tâches
- Afficher un message lorsque la liste est vide
- Ajouter une tâche avec la touche Entrée
- Supprimer toutes les tâches avec une confirmation
- Sauvegarder automatiquement les tâches dans le localStorage

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- localStorage

## Compétences travaillées

Ce projet m'a permis de pratiquer plusieurs notions importantes :

- Structuration d'une page HTML
- Mise en forme avec CSS
- Utilisation de Flexbox
- Manipulation du DOM avec JavaScript
- Création dynamique d'éléments HTML
- Gestion des événements avec addEventListener
- Utilisation des tableaux et objets JavaScript
- Utilisation de forEach et filter
- Gestion de données avec localStorage
- Mise à jour dynamique de l'interface utilisateur

## Lancer le projet

Pour lancer le projet, il suffit d'ouvrir le fichier `index.html` dans un navigateur.

Il est aussi possible d'utiliser l'extension Live Server dans Visual Studio Code pour lancer le projet en local.
